## 🎯 目標玩家定位

| 玩家動機 | 強度 | 核心依據 |
|:---|:---|:---|
| **收集驅動** | 🟣🟣🟣🟣🟣 極高 | 100+ Tatari 角色、4 階進化、閃光型態，收集癮極強 |
| **視覺享受** | 🟣🟣🟣🟣🟣 極高 | 「cute」「adorable」「art style」為最高頻正面關鍵字，美術是核心吸引力 |
| **休閒放鬆** | 🟣🟣🟣🟣 高 | 「chill」「relaxing」「time killer」「anxiety relief」高頻出現 |
| **博奕刺激** | 🟣🟣🟣🟣 高 | 彈珠台 = 柏青哥 / Plinko，本質為隨機獎勵機制，「gambling」字眼高頻出現 |
| **策略挑戰** | 🟣🟣🟣 中 | 屬性剋制、陣型搭配有一定策略深度，但 auto-battle 降低門檻 |
| **社交互動** | 🟣🟣 低 | 有世界聊天與 PvP 突襲，但聊天室管理不善反成負面因素 |

**主要受眾：Z 世代休閒玩家**（15–28 歲，男女各半偏女性）
- 被可愛美術吸引、喜歡寶可夢式收集但不想花太多時間
- 習慣看 YouTube Shorts / TikTok 發現新遊戲
- 對「無廣告」敏感度高，願意嘗試 F2P 但排斥被逼付費

**次要受眾：PvZ / 塔防老玩家**（25–40 歲，偏男性）
- 被 PvZ 懷舊感吸引，但期待更多深度
- 對 auto-battle 和彈珠台機制感到新鮮

玩家組成：收集型 ~40%、休閒型 ~30%、博奕型 ~20%、策略型 ~10%。

### 品類定位

| 層級 | 分類 |
|:---|:---|
| **大品類** | 休閒遊戲（Casual） |
| **子品類** | 寵物收集 × 塔防（Creature Collection × Tower Defense） |
| **細分品類** | Hybridcasual — 休閒入口 + 中核深度（Idle TD + Gacha Collection + Pachinko） |

### 核心玩法（Core Game）

- **車道塔防**：PvZ 式橫向車道戰鬥，將收集的 Tatari 放置在車道上防禦 Zobo（殭屍）波次
- **屬性剋制**：火 > 草 > 水 > 火等屬性相剋，需根據敵方屬性調整陣型
- **自動 / 手動部署**：支援 Quick Deploy 自動部署 + 手動擺放雙模式
- **彈珠台（Pachinko）**：核心升級機制，發射彈珠獲得經驗值、資源、新 Tatari，所有角色共享等級
- **營地經營**：佈置個人營地、與 Tatari 互動（餵食、泡水）、種植彈珠果樹（每 2 小時收穫）
- **劇情推進**：關卡分為主線世界、Boss 戰、道場挑戰，多種遊戲模式交替

### 養成系統（Meta Game）

| 系統 | 有無 | 說明 |
|:---|:---|:---|
| **角色收集** | ✅ | 100+ Tatari，各有獨特外觀、屬性與技能 |
| **進化系統** | ✅ | 4 階進化（需重複角色碎片），解鎖閃光（Glitter）特殊型態 |
| **等級共享** | ✅ | 所有 Tatari 共享帳號等級，新角色自動同步等級（降低養成壓力） |
| **PvP 突襲** | ✅ | 可攻擊其他玩家營地搶奪資源，但引發部分玩家反感 |
| **共鬥模式** | ✅ | Co-op Horde 生存模式，多人協力防禦 |
| **卡牌系統** | ⚠️ | 存在但部分玩家反映過於複雜、說明不足 |
| **深度社交** | ⚠️ | 世界聊天存在但管控薄弱，多名玩家反映充斥不當言論 |
| **每日任務** | ⚠️ | 有基本每日獎勵，但玩家反映彈珠獲取不足 |
| **公會系統** | ❌ | 目前未發現公會 / 社團系統 |
| **賽季系統** | ❌ | 上線 5 天，尚未觀察到賽季循環 |

### 變現設計

| 設計 | 評估 |
|:---|:---|
| **無廣告** | ✅ 零外部廣告，玩家極度好評。在手遊市場中是罕見的差異化優勢 |
| **通行證（Weekly / Monthly Pass）** | ✅ 價格合理（週卡 / 月卡），提供穩定長期價值 |
| **新手禮包** | ✅ $5 起跳的 Starter Pack，玩家評價「物超所值」 |
| **閃耀果實（Glitter Fruit）** | ⚠️ 高級貨幣，用於角色突變 / 特殊進化，存在 P2W 隱憂 |
| **限時禮包彈窗** | 🔴 **最大爭議點**：每次切換畫面就彈出 2-4 個內購彈窗，嚴重影響體驗。多名 4★→2★ 降評均因此而起 |
| **彈珠台綁定變現** | ⚠️ 彈珠為核心資源，初期大量贈送但後期稀缺，形成「甜蜜期→付費牆」的典型 F2P 漏斗 |

## ⚠️ 關鍵風險

- 🚩 **內購彈窗侵略性過高**：9.5% 評論提及，「切換頁面就彈 2-4 個購買視窗」為最高頻負面主題，已造成實際降評（5★→2★），若不修正將持續傷害留存
- 🚩 **甜蜜期→付費牆斷崖**：初期大量彈珠 → 數小時後資源枯竭，多名玩家表示「前幾小時很棒但之後無事可做」，D7 留存率堪憂
- 🚩 **聊天室安全問題**：4.2% 評論提及，世界聊天無有效過濾機制，已出現針對未成年人的不當言論，存在合規與公關風險
- 🟡 **誤導性廣告反噬**：部分買量素材展示合成 / merge 玩法，與實際遊戲不符，導致預期落差與低評分
- 🟡 **長線內容不足**：上線版本僅 1 個世界（World 1），通關後缺乏內容支撐，道場 / Boss 有限，依賴快速更新填充
- 🟡 **Auto-battle 降低參與感**：自動戰鬥便利但削弱策略感，部分玩家認為「不像在玩遊戲」

## 💡 重點

- ✅ **可參考**：高品質非 AI 手繪美術在當前市場具強烈差異化，美術本身就是買量素材，自帶自然流量傳播力
- ✅ **可參考**：塔防 × 收集 × 彈珠台的「Hybridcasual」品類混搭符合 2026 混合休閒趨勢，等級共享設計降低養成焦慮
- ✅ **可參考**：「零廣告」策略在廣告氾濫的手遊市場中成為玩家自發傳播的口碑武器
- ❌ **要避免**：內購彈窗洗臉式設計（每次切換頁面彈 2-4 個），已造成大量降評，應改為首次啟動展示 + 商店入口
- ❌ **要避免**：甜蜜期過短的 F2P 漏斗（3-5 小時後斷崖式資源稀缺），轉化率未必提升但必定傷害留存
- ❌ **要避免**：在受眾含大量未成年的遊戲中不設聊天文字過濾，已出現針對未成年人不當言論，存在合規風險
- 📌 **結論**：**短線爆發力極強、長線健康度待觀察的品類混搭黑馬。** 莉莉絲以頂級美術 + 品牌力 + UA 投放製造了完美的上線爆點，但彈窗侵略性和進度牆設計正在以肉眼可見的速度消耗好感。若 2-4 週內不修正變現策略，高評分和高排名可能快速滑落。**值得深度研究，持續追蹤。**
