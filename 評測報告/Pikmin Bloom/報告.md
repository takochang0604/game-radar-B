| 玩家動機 | 強度 | 核心依據 |
|:---|:---|:---|
| **收集欲 / 完成感** | 🟣🟣🟣🟣🟣 極高 | 150+ 種 Decor Pikmin，集齊套裝是核心驅動力 |
| **健康 / 運動動機** | 🟣🟣🟣🟣🟣 極高 | 大量用戶提及「因為它才開始每天走路」 |
| **治癒感 / 放鬆** | 🟣🟣🟣🟣 高 | 「cute」「adorable」「relaxing」是最高頻詞彙 |
| **探索欲** | 🟣🟣🟣 中 | 明信片系統鼓勵到訪新地點 |

- 被 Pikmin 4 帶入，持續收集稀有 Decor 套裝

## 🎮 產品解構分析

> 評測依據：GameRefinery 功能解構法（三層分類）

### 品類定位

| 層級 | 分類 |
|:---|:---|
| **大品類** | LBS / AR（Location-Based Service） |
| **子品類** | 散步養成 / 收集 |
| **細分品類** | 步數驅動 × 角色收集 × 花卉種植 |

> 💡 與 Pokémon GO 同為 LBS 品類，但核心差異在於**完全移除 PvP 壓力**，定位為「生活伴侶型 App」而非「遊戲」。

### 核心玩法（Core Game）

```
走路（步數累積）
  ↓
育苗（步數孵化 Pikmin）→ 收集 Decor Pikmin
  ↓
餵食花蜜 → 採集花瓣
  ↓
種花（地圖上種植花卉）→ 獲得金幣
  ↓
蘑菇戰鬥（協力）→ 獲得水果 → 更多花蜜
  ↓
探險（Pikmin 自動外出撿回物品 / 明信片）
```

**設計精髓**：
- **極低注意力需求**：設定後口袋裡自動計步，不需盯螢幕
- **正向回饋感**：走路 → 看到花田在地圖上綻放，視覺回饋即時
- **無廣告**：完全零廣告設計，在免費遊戲中極為罕見

### 養成系統（Meta Game）

| 系統 | 深度 | 說明 |
|:---|:---|:---|
| Pikmin 收集 | 🔴🔴🔴🔴 高 | 150+ 種 Decor，每月新增活動限定款 |
| 花卉種植 | 🔴🔴 中 | 多種花卉，隱私區域保護設計 |
| 蘑菇戰鬥 | 🔴🔴 中 | 簡易協力系統，Pikmin 屬性配隊 |
| Mii 角色 | 🔴 低 | 服裝自訂，但深度有限 |
| 生活日誌 | 🔴🔴 中 | 每日步數 + 心情記錄 + 回顧 |

### 變現設計

| 設計 | 評估 |
|:---|:---|
| Pikmin 倉庫擴充 | ✅ 剛需型消費，核心付費點（$2.99/次） |
| Mii 服裝 | ✅ 外觀消費，不影響遊戲平衡 |
| 活動通行證 | ✅ 收集控的 FOMO 驅動，但可 F2P 完成 |
| 蘑菇戰鬥票券 | ⚠️ 每日限 3 場，額外需購買，部分用戶抱怨 |
| 金幣系統 | ✅ 可透過種花免費獲得（60 幣/天上限） |
| 廣告 | ✅ **完全零廣告** |
| 直營網路商城（2024 新增） | ✅ 繞過平台抽成，提升營收 |

> ✅ **變現健康度高**：無 PvP 壓力、無廣告、無付費牆。核心消費為倉庫擴充與收藏品，符合「gym membership」心理模型。

## ⚠️ 關鍵風險

| 緊急程度 | 風險項目 | 潛在影響 |
|:---|:---|:---|
| 🔴 高 | **效能問題（耗電/過熱/Lag/閃退）** | 多名長期用戶因效能降低評分或停玩；社群 2026/03 發起效能更新請願（上次優化為 2023/05） |
| 🔴 高 | **步數計算不準確** | 核心機制的信任基礎，Android 智慧手錶同步尤其嚴重 |
| 🟡 中 | **Scopely 收購後變現壓力** | 已出現部分用戶質疑「IAP 越來越多」；商業化加速可能打破現有「零壓力」體驗平衡 |
| 🟡 中 | **蘑菇戰鬥競爭加劇** | 用戶增長後，免費玩家難以搶到蘑菇戰鬥名額，需購買票券 |
| 🟡 中 | **Spoofer / 作弊行為** | 破壞公平性，影響花田與蘑菇戰鬥體驗 |
| 🟢 低 | **Niantic Spatial 技術依賴** | AR 平台與地圖資料的持續授權是否受收購影響 |

## 💡 重點

- ✅ **可參考**：「零廣告 + 零 PvP + 極低注意力」三重減法設計，在 LBS 品類中成功建立差異化護城河；變現專注於收集欲＋便利性，而非競爭焦慮
- ✅ **可參考**：IP 聯動策略（Pikmin 4 → Bloom）有效帶動下載量翻倍，跨平台角色轉移提升黏著度
- ⚠️ **需觀察**：Scopely 接手後能否維持「不壓榨用戶」的品牌調性——目前 IAP 增加已引發部分用戶疑慮
- ⚠️ **需觀察**：效能優化的優先級——社群已明確表達這是最大痛點，上次優化距今超過 3 年
- ❌ **要避免**：在低壓力產品中引入高壓力變現（如 FOMO 活動 + 稀有度 RNG），會直接損害核心價值主張
- 📌 **結論**：**穩健長線型典範產品。** 以減法設計在 LBS 紅海中找到藍海定位。營收 $1 億+ 證明「健康變現」可以永續。最大風險不是競品，而是自身效能退化與收購後的變現壓力。
