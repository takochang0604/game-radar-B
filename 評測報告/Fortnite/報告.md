## 🎯 目標玩家定位

| 玩家動機 | 強度 | 核心依據 |
|:---|:---|:---|
| **競爭 / 勝負慾** | 🟣🟣🟣🟣🟣 極高 | Battle Royale 核心機制：100 人淘汰賽，Ranked 排位系統，電競錦標賽 |
| **社交 / 共玩** | 🟣🟣🟣🟣🟣 極高 | 55 則評論（14.8%）提及朋友共玩，派對系統、語音聊天、跨平台組隊 |
| **收集 / 外觀** | 🟣🟣🟣🟣 高 | 1,000+ 種造型，IP 聯動皮膚（Marvel、Star Wars、Overwatch），Battle Pass 季度收集 |
| **探索 / 創作** | 🟣🟣🟣🟣 高 | UEFN 創作平台提供數千張社群地圖：大亨、死亡跑酷、殭屍生存等 |
| **休閒 / 放鬆** | 🟣🟣🟣 中 | Zero Build 模式、LEGO Fortnite、Fortnite Festival 音樂模式降低壓力 |


**主要受眾：核心射擊玩家**（13–25 歲，男性為主 ~70%）
- 追求高強度 PvP 對抗，喜歡建造+射擊的獨特戰鬥節奏
- PC/主機玩家在外出時透過手機延續遊戲進度

**次要受眾：社交休閒玩家**（10–18 歲，男女均衡）
- 被 IP 聯動皮膚吸引（Marvel、Overwatch 等），以社交為主要動機
- 偏好 Zero Build、Creative 地圖等低壓力模式

玩家組成：競技型 ~40%、社交型 ~35%、收集型 ~15%、創作型 ~10%。

### 品類定位

| 層級 | 分類 |
|:---|:---|
| **大品類** | 動作（Action） |
| **子品類** | 大逃殺（Battle Royale） |
| **細分品類** | 建造+射擊 混合型 BR × UGC 創作平台 |

> 💡 Fortnite 已從單純的 BR 遊戲進化為「遊戲元宇宙平台」，整合了 BR、Zero Build、LEGO 生存、音樂節、UGC 創作等多種體驗。

### 核心玩法（Core Game）

- **建造系統**：牆壁/斜坡/地板即時建造，為 BR 品類中唯一的建造+射擊混合機制，創造獨特的立體空間戰鬥
- **Zero Build**：2022 年推出的無建造模式，移除建造機制改用 Overshield，大幅降低入門門檻，成為行動端主流模式
- **IP 聯動生態**：業界最大規模的 IP 合作體系（Marvel、Star Wars、Dragon Ball、Overwatch、Nike 等），每季帶來新話題
- **UEFN 創作平台**：基於 Unreal Engine 的 UGC 創作工具，社群已產出數千張地圖，形成自給自足的內容生態

### 養成系統（Meta Game）

| 系統 | 有無 | 說明 |
|:---|:---|:---|
| **Battle Pass** | ✅ | 每季 $8，100+ 層獎勵，核心留存機制 |
| **Ranked 排位** | ✅ | 青銅→鑽石→Unreal 分段，電競路線 |
| **皮膚收藏** | ✅ | 1,000+ 種造型，IP 聯動限定款驅動 FOMO |
| **LEGO Fortnite** | ✅ | 獨立的開放世界生存建造模式 |
| **Fortnite Festival** | ✅ | 音樂節奏遊戲模式，Music Pass 季度內容 |
| **Creative/UEFN** | ✅ | 社群 UGC 平台，大亨/死亡跑酷/殭屍生存等數千張地圖 |

### 變現設計

| 設計 | 評估 |
|:---|:---|
| **V-Bucks 虛擬貨幣** | ✅ 純外觀消費，不影響遊戲平衡，業界最健康的微交易模型 |
| **Battle Pass（$8/季）** | ✅ 高 CP 值，免費玩家也能獲得部分獎勵 |
| **外觀商店（每日輪替）** | ✅ FOMO 驅動但純外觀，無 P2W |
| **Crew 訂閱（$12/月）** | ✅ 包含 Battle Pass + 每月限定皮膚 + 1,000 V-Bucks |
| **廣告** | ✅ **完全零廣告** |
| **開箱/轉蛋** | ✅ **完全沒有隨機抽獎機制**，所有商品明碼標價 |

> ✅ **變現健康度業界頂級**：零廣告、零 P2W、零開箱、純外觀。年營收 $60 億證明「不賣數值」也能成為史上最賺錢遊戲之一。

## ⚠️ 關鍵風險

- 🚩 **行動端效能優化不足**：14% 的用戶評論提及延遲、低幀率、畫面模糊，尤其中低端裝置體驗差距明顯，已開始影響 Google Play 評分（3.8 vs iOS 4.4）
- 🚩 **下載/更新容量巨大**：基礎安裝 6-8 GB + 頻繁的大型更新，在儲存空間有限的新興市場（東南亞）形成進入障礙
- 🟡 **跨平台操作不平衡**：觸控 vs 手把 vs 鍵鼠的技術差距讓行動端玩家在跨平台對戰中處於結構性劣勢
- 🟡 **V-Bucks / Battle Pass 漲價爭議**：社群出現「付一樣的錢得到更少」的批評，長期可能損害付費意願
- 🟡 **Epic v Apple 訴訟持續中**：澳洲仍然無法上架，未來若出現新的法律爭議可能再次影響可用性

## 💡 重點

- ✅ **可參考**：純外觀變現模式（零廣告 + 零 P2W + 零開箱）年收 $60 億，證明「不賣數值」也能成為史上最賺錢遊戲之一
- ✅ **可參考**：Zero Build 模式成功擴大受眾面——降低入門門檻不代表降低遊戲深度，而是為不同玩家提供選擇
- ✅ **可參考**：UEFN 創作平台讓社群產出內容並分潤，建立自給自足的 UGC 生態，大幅延長產品生命週期
- ⚠️ **需觀察**：行動端效能優化能否跟上 PC/主機版本的更新速度——目前是最大用戶痛點
- ❌ **要避免**：跨平台配對不平衡問題——觸控 vs 手把 vs 鍵鼠的技術差距會讓行動端玩家感到不公
- 📌 **結論**：**現象級長線平台型產品。** 不再只是一款遊戲，而是結合了 BR、創作、社交、音樂的「遊戲元宇宙」。回歸 iOS 後立即稱霸多國排行，證明品牌號召力依然頂級。唯一短板是行動端效能優化。
