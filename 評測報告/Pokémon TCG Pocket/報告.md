## 🎯 目標玩家定位

| 玩家動機 | 強度 | 核心依據 |
|:---|:---|:---|
| **收集慾望** | 🟣🟣🟣🟣🟣 極高 | 開包抽卡是核心體驗，稀有卡（Crown/Immersive）驅動強烈收集衝動 |
| **懷舊情感** | 🟣🟣🟣🟣🟣 極高 | Pokémon IP 橫跨 30 年，覆蓋 8-40 歲多世代玩家群體 |
| **社交展示** | 🟣🟣🟣🟣 高 | 卡牌展示、交易、分享功能完善，收藏品屬性強 |
| **休閒打發** | 🟣🟣🟣 中 | 每局 5-10 分鐘，適合碎片時間，但對戰深度有限 |
| **策略競技** | 🟣🟣 低 | 20 張牌組 + 翻硬幣機制，策略性低於傳統 TCG |

**主要受眾：寶可夢粉絲 + 卡牌收集愛好者**（15-35 歲，男女均衡）
- 對 Pokémon IP 有情感連結，童年看動畫 / 玩 Game Boy
- 享受開包「賭運」的刺激感和稀有卡的成就感
- 不一定是重度卡牌遊戲玩家

**次要受眾：休閒手遊玩家**
- 被 Pokémon 品牌吸引入坑
- 以每日開包為主要活動，對戰參與較少

玩家組成：收集型 ~50%、休閒型 ~30%、競技型 ~20%。

### 品類定位

| 層級 | 分類 |
|:---|:---|
| **大品類** | 卡牌遊戲（Card Game） |
| **子品類** | 集換式卡牌（Trading Card Game） |
| **細分品類** | IP 授權卡牌收集 + 簡化對戰 |

### 核心玩法（Core Game）

- **開包抽卡**：每日免費 2 包（12 小時冷卻），開包動畫精美，稀有卡有沉浸式全幅展示效果，是遊戲最核心的爽感來源
- **卡牌對戰**：20 張精簡牌組，比傳統 TCG 快速。先手/後手由翻硬幣決定，單局 5-10 分鐘
- **Wonder Pick**：從其他玩家開出的包中隨機抽一張卡，社交式 Gacha 變體
- **收藏展示**：收集不同稀有度卡牌（Normal → Rare → Immersive → Crown），可自定展示牆分享
- **Solo 戰鬥**：AI 對手練習模式，含 Expert 高難度挑戰

### 養成系統（Meta Game）

| 系統 | 有無 | 說明 |
|:---|:---|:---|
| **卡牌收集圖鑑** | ✅ | 按擴充包分類，含完成度追蹤 |
| **牌組構築** | ✅ | 20 張牌組限制，可存多組方案 |
| **排位賽系統** | ✅ | PvP 段位制，含 Emblem 活動獎勵 |
| **Pack Points** | ⚠️ | 開包累積點數可兌換指定卡，但不跨擴充包 |
| **Trade / 交易** | ✅ | 1.4 版新增，可與好友交換卡牌 |
| **角色成長** | ❌ | 無等級、裝備等傳統養成元素 |

### 變現設計

| 設計 | 評估 |
|:---|:---|
| **Gacha 開包** | ⚠️ 核心營收來源。每日免費 2 包限制創造付費動力，但 Crown 卡天花板極高（韓國用戶反映 ~40 萬日圓） |
| **Premium Pass（$9.99/月）** | ✅ 價格合理，每日多 1 包 + 獨家商品，14 天免費試用降低門檻 |
| **Poké Gold** | ⚠️ 直購虛擬貨幣加速開包，加劇 F2P 與付費玩家差距 |
| **Pack Hourglasses** | 🔴 付費加速機制，本質上是「花錢跳過等待時間」，引發 P2W 爭議 |
| **無廣告** | ✅ 無任何廣告植入，維持高端品牌感 |

### 策略觀察

1. **多元化 IP 授權策略**：Pokémon 將行動遊戲分散給 5+ 家合作開發商，每家專攻不同品類（AR/卡牌/MOBA/RPG/休閒），降低單一產品風險
2. **DeNA 深度綁定**：透過成立子公司 Pokémon Card D Studio 專門開發 TCG Pocket，顯示長期投入而非短期合作
3. **擴充包驅動型營運**：約每 4-6 週推出新擴充包（Genetic Apex → Mythical Island → Space-Time Smackdown → Pulsing Aura → Paradox Drive），形成穩定的營收脈衝週期

## ⚠️ 關鍵風險

- 🚩 **營收過度依賴擴充包脈衝**：營收高度集中在新包上線的 3-5 天內，包間低谷期排名快速回落（如 iOS 營收從 #1 回落到 #25+），長線營收穩定性存疑
- 🚩 **用戶疲勞信號明顯**：25.8% 的 1 星評價率偏高，多名長期玩家反映 pull rate 感覺被調低、UI 越來越慢、新鮮感消退
- 🚩 **技術債務累積**：隨擴充包增加，遊戲容量膨脹至 3-5GB，載入速度和閃退問題加劇，多則評論提及效能下降
- 🟡 **Pack Points 不跨包爭議**：每次新擴充包上線，舊包的 Pack Points 無法沿用，迫使玩家從零開始累積，引發 P2W 批評
- 🟡 **對戰深度不足**：20 張牌組 + 翻硬幣機制限制了策略深度，難以留住核心 TCG 玩家，可能導致玩家群體向「開包即走」靠攏

## 💡 重點

- ✅ **可參考**：脈衝式擴充包更新週期（每 4-6 週）精準刺激回流消費，是卡牌品類的最佳營運節奏參考
- ✅ **可參考**：開包動畫和稀有卡展示效果的極致打磨，是開包爽感 = 付費動力的最佳範例
- ✅ **可參考**：20 張精簡牌組 + 5-10 分鐘單局，成功將傳統 TCG 適配行動裝置碎片時間
- ❌ **要避免**：Pack Points 不跨擴充包的設計引發強烈負評，消耗玩家信任
- ❌ **要避免**：過度依賴 IP 吸引力忽視對戰深度，導致核心 TCG 玩家流失
