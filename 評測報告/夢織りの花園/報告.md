### 核心機制

1. **花卉經營循環**：種植 → 收穫（一鍵）→ 獲得資源 → 解鎖稀有花種與限定色系
2. **妖精養成 Meta**：花園成長解鎖妖精新型態與能力，提供長期養成目標
3. **裝扮系統**：角色換裝（休閒→精靈風）+ 花瓶/裝飾品自由搭配
4. **變現機制**：IAP（虛擬貨幣/裝扮道具）+ Rewarded Ads + 月卡去廣告訂閱
5. **社交/行銷**：「虛實連結」——遊戲活動兌換真實鮮花寄送到家

### 與市場領導者差異

| | 旅かえる（旅行青蛙） | 夢織りの花園 |
|---|---|---|
| **互動頻率** | 極低（準備行囊等回家） | 中等（種花、經營、佈置） |
| **養成深度** | 淺（收集照片名產） | 深（妖精育成+裝扮+花園設計） |
| **變現強度** | 極低（幾乎無 IAP） | 中高（IAP+廣告+訂閱混合） |
| **長期留存策略** | 隨機驚喜驅動 | 目標導向+收集驅動 |
| **社群傳播** | 分享青蛙照片 | 分享花園+真花活動 |
